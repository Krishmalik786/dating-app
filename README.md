# Dating App Match Predictor

A machine learning project that predicts dating app match outcomes from user profile behaviour, paired with an AI-powered explanation layer built on Groq (LLaMA 3.3).

---

## Quickstart

**1. Clone and install dependencies**
```bash
git clone <repo-url>
cd dating-app
python -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

**2. Add your Groq API key**
```bash
# Create a .env file in the project root
echo "GROQ_API_KEY=your_key_here" > .env
```
Get a free key at [console.groq.com](https://console.groq.com).

**3. Run the notebook to train models**

Open `notebook/dating_match_predicition.ipynb` and run all cells top to bottom. This generates all files in `models/`.

**4. Launch the app**
```bash
streamlit run app.py
```

---

## What the Notebook Does

| Section | Description |
|---|---|
| Data Loading & EDA | Loads 50k rows, checks distributions, plots correlations |
| Feature Engineering | 49 interest-tag flags, ordinal encoding, log transforms, cyclical time encoding, polynomial features — 160+ total features |
| Preprocessing | StandardScaler on continuous features, stratified 80/20 split |
| Model Training | XGBoost, LightGBM, CatBoost, Random Forest, MLP, Stacking Ensemble |
| Evaluation | Accuracy, Weighted F1, 3-fold stratified CV, confusion matrix, feature importance |
| Sample Predictions | Two test profiles run through the full pipeline |
| AI Explanation | Groq (LLaMA 3.3-70b) generates personalised match insights |

## Models Trained

| Model | Test Accuracy | Weighted F1 |
|---|---|---|
| Stacking Ensemble (XGB + LGB + MLP → LR) | 10.44% | 0.074 |
| CatBoost | 9.86% | 0.097 |
| XGBoost | 9.84% | 0.098 |
| Random Forest | 9.69% | 0.097 |
| MLP (Neural Net) | 9.36% | 0.093 |
| LightGBM | 9.32% | 0.093 |

> **Why ~10% accuracy?** The dataset is synthetically generated with randomly assigned `match_outcome` labels. Mutual information between every feature and the target is near zero (highest: MI = 0.005). All models are performing at the theoretical ceiling for random labels across 10 balanced classes. See [`model_selection.md`](model_selection.md) and [`research_note.md`](research_note.md) for full analysis.

## Streamlit App

The app allows a user to fill in their dating profile and receive:
- A predicted match outcome from any of the 5 trained models
- A confidence score and top-5 probability breakdown
- An AI-generated explanation with profile strengths and improvement tips

A **model selector** in the sidebar lets you switch between all trained models and compare results live.
